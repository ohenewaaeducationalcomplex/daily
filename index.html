IndexedDB Storage Information

This system uses IndexedDB, which can store up to 50% of available disk space (typically several GB).

Each member profile picture is compressed to ~50KB, allowing storage for 1000+ members with photos.

  • Maximum storage: Up to several GB (browser dependent)
  • Supports 1000+ members with profile pictures
  • All data persists between browser sessions
  • Data is stored locally, no internet required
  • Backup and restore functionality included
